# Exercise Library

## Movement Pattern Taxonomy

Every resistance exercise fits into one of these primary patterns. When substituting exercises, swap within the same pattern to preserve training stimulus.

| Pattern | Primary Muscles | Key Exercises |
|---|---|---|
| Horizontal push | Chest, front delts, triceps | Bench press, dumbbell press, push-ups, dips |
| Vertical push | Shoulders, triceps, upper chest | Overhead press, dumbbell shoulder press, landmine press |
| Horizontal pull | Back (thickness), rear delts, biceps | Barbell row, dumbbell row, cable row, T-bar row |
| Vertical pull | Back (width), biceps | Pull-ups, lat pulldown, cable pullover |
| Hip hinge | Posterior chain (hamstrings, glutes, erectors) | Deadlift, RDL, stiff-leg deadlift, hip thrust |
| Knee dominant | Quads, glutes | Squat, leg press, lunge, Bulgarian split squat |
| Isolation -- upper | Biceps, triceps, delts, forearms | Curls, pushdowns, lateral raises, face pulls |
| Isolation -- lower | Quads, hamstrings, calves, adductors | Leg extension, leg curl, calf raise, adductor machine |

## Compound Lifts -- Key Cues & Common Errors

### Barbell Back Squat (Agachamento)

**Setup**: bar on upper traps (high bar) or rear delts (low bar), feet shoulder-width, toes slightly out.

| Phase | Key Cue | Common Error |
|---|---|---|
| Descent | "Sit between your knees" | Knees caving inward (valgus) |
| Bottom | "Chest up, elbows under bar" | Excessive forward lean / butt wink |
| Ascent | "Drive the floor away" | Hips shooting up first (good morning squat) |
| Bracing | "Big breath, brace 360 degrees" | Only bracing the front (no oblique engagement) |

**Depth target**: hip crease below knee (parallel or below). If mobility limits depth, work on ankle dorsiflexion and hip mobility.

### Conventional Deadlift (Peso Morto)

**Setup**: feet hip-width, bar over mid-foot, grip just outside knees, hips between shoulders and knees.

| Phase | Key Cue | Common Error |
|---|---|---|
| Setup | "Push the floor away with your legs" | Jerking the bar (arms should be passive hooks) |
| Mid-pull | "Bar stays on the legs" | Bar drifting forward away from the body |
| Lockout | "Squeeze glutes, stand tall" | Hyperextending the lower back |
| Descent | "Push hips back first, then bend knees" | Rounding the back on the way down |

### Barbell Bench Press (Supino)

**Setup**: 5 points of contact (head, upper back, glutes on bench; both feet on floor). Retract and depress scapulae.

| Phase | Key Cue | Common Error |
|---|---|---|
| Unrack | "Straight arms, pull bar over chest" | Flaring elbows immediately |
| Descent | "Pull the bar to your chest with your lats" | Bouncing off chest |
| Touch point | Lower sternum / nipple line | Touching too high (neck) or too low (stomach) |
| Press | "Push yourself into the bench" | Losing scapular retraction (shoulders rolling forward) |
| Arch | Slight natural arch, not extreme | Flat back (loses stability) or excessive arch (risky) |

### Overhead Press (Press Militar)

**Setup**: bar in front rack, elbows slightly in front of bar, feet hip-width.

| Phase | Key Cue | Common Error |
|---|---|---|
| Press | "Push your head through the window" | Pressing the bar forward instead of straight up |
| Lockout | "Ears in front of arms at lockout" | Leaning back excessively (turns into incline bench) |
| Core | "Squeeze glutes, brace abs" | Hyperextending the lower back under load |

### Barbell Row (Remada com Barra)

**Setup**: hip hinge to ~45 degrees torso angle, bar hanging at arm's length, overhand or underhand grip.

| Phase | Key Cue | Common Error |
|---|---|---|
| Pull | "Drive elbows toward the ceiling" | Using momentum / swinging torso |
| Squeeze | "Squeeze a pencil between your shoulder blades" | Not fully retracting scapulae |
| Torso | "Maintain your hip hinge angle" | Rising up during the pull (cheat rows) |

### Romanian Deadlift / Stiff-Leg Deadlift (Stiff)

**Setup**: bar in hand at lockout, soft knee bend (RDL) or straighter legs (stiff-leg), hip hinge pattern.

| Phase | Key Cue | Common Error |
|---|---|---|
| Descent | "Push your hips back like closing a car door" | Bending the knees too much (turns into deadlift) |
| Stretch | "Feel the hamstrings stretch, not the lower back" | Rounding the lower back to get more range |
| Ascent | "Drive hips forward, squeeze glutes" | Using the lower back to pull up |
| Bar path | "Bar slides down the front of your legs" | Bar drifting forward |

## Isolation Exercises -- Key Cues

### Lateral Raise (Elevacao Lateral)

- Lead with the elbows, not the hands
- Slight forward lean to keep tension on the side delt
- Lift to shoulder height, not above (above = traps take over)
- Control the eccentric (don't just drop the weight)
- Pinky slightly higher than thumb at the top (old cue; actually just keep a neutral wrist)

### Bicep Curl (Rosca Biceps)

- Elbows pinned to sides (no forward swing)
- Full extension at the bottom (fight the urge to shortcut ROM)
- Supinate (turn pinky outward) at the top for peak contraction with dumbbells
- Don't lean back to cheat the weight up

### Tricep Pushdown (Extensao Triceps)

- Elbows pinned to sides
- Only the forearms move
- Full lockout at the bottom (squeeze)
- Controlled eccentric back to ~90 degrees
- Don't let the weight stack pull your elbows forward

### Leg Extension (Extensao de Pernas)

- Adjust pad to sit just above the ankle
- Full extension at the top (squeeze the quad)
- Slow eccentric (3 seconds minimum for hypertrophy stimulus)
- Don't use momentum to swing the weight up

### Leg Curl (Flexao de Pernas)

- Lying or seated -- both are valid; seated provides more hamstring stretch
- Full contraction (heel toward glute)
- Don't lift hips off the pad (lying variation)
- Slow eccentric for maximal stretch under load

## Substitution Chains

When an exercise needs to be swapped (equipment unavailable, injury, plateau), use the same movement pattern. Listed from most similar to least:

### Horizontal Push
```
Barbell bench press
  -> Dumbbell bench press (more ROM, stability demand)
  -> Machine chest press (isolates, less stability)
  -> Push-ups (bodyweight, add bands/vest for load)
  -> Cable flye (isolation, stretch emphasis)
```

### Vertical Push
```
Barbell overhead press
  -> Dumbbell shoulder press (seated or standing)
  -> Landmine press (easier on shoulders)
  -> Machine shoulder press
  -> Cable lateral raise (isolation alternative)
```

### Horizontal Pull
```
Barbell bent-over row
  -> Dumbbell row (unilateral, less spinal load)
  -> Cable seated row (constant tension)
  -> T-bar row (chest-supported = less fatigue)
  -> Machine row (guided, good for burnout sets)
```

### Vertical Pull
```
Pull-ups (weighted if possible)
  -> Lat pulldown (adjustable load)
  -> Cable pullover (isolates lats, no bicep)
  -> Machine pulldown
  -> Band-assisted pull-ups (progression tool)
```

### Hip Hinge
```
Conventional deadlift
  -> Sumo deadlift (more quad, less back)
  -> Trap bar deadlift (more quad, easier on back)
  -> Romanian deadlift (hamstring emphasis)
  -> Hip thrust (glute emphasis, less spinal load)
  -> Cable pull-through (lighter alternative)
```

### Knee Dominant
```
Barbell back squat
  -> Front squat (more quad, more core)
  -> Leg press (less spinal load)
  -> Bulgarian split squat (unilateral, stability)
  -> Goblet squat (mobility work, lighter loads)
  -> Hack squat machine
```
